# Snake Game (Console Edition)

A classic Snake game implemented in **C++** for the **Windows Console**.

## Features

- **Classic Snake gameplay**: control the snake using the arrow keys.
- **Score system**: earn points by eating food.
- **Lives system**: start with 3 lives. Lose a life when colliding with walls or yourself.
- **Extra life pickup**: collect +1 ups if your lives are below 3.
- **Visual feedback**:
  - Flashes **red** on losing a life.
  - Flashes **green** when collecting a life.
- **Game over screen**: displays final score below the game grid.
- **Object-oriented design**: uses classes, inheritance, and smart pointers.
- **Console-based rendering**: no external libraries required, runs entirely in the Windows console.

## Controls

- **Arrow keys**: Move the snake  
- **Q**: Quit the game
